# Events Gallery Images

Add your event photos here to display them in the Events Gallery section.

## Expected Files:
- `hackerrank-event-1.jpg` - HackerRank Ambassador event photo
- `meetup-1.jpg` - Tech meetup and networking event photo
- `workshop-1.jpg` - Workshop event photo

You can add more event images by:
1. Adding the image files to this directory
2. Updating the `events` array in `/src/pages/Events.tsx` with the new image paths

## Image Guidelines:
- Recommended format: JPG or PNG
- Recommended size: 1200x800px (3:2 aspect ratio)
- File naming: Use descriptive names in lowercase with hyphens (e.g., `tech-conference-2025.jpg`)
